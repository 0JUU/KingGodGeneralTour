# 킹갓제너럴투어

### ◎ 기획 의도
  - 최근 거리두기 폐지와 자가격리 의무 해제로 인한 여행 수요 급증
  - 이에 반해 항공권 예약 사이트의 국가별 방역 정보 안내 미흡 및 주변 여행지에 대한 정보 없음<br>
→ 좀더 정확한 방역 정보를 보여주며, 주변 여행지 정보를 추가한 국내/외 항공권 예약 사이트를 구상함
<br/><br/>

### ◎ 구현 기능
  #### 1. 공지사항 <br>
    - 긴급공지 상단 고정
    - 이전글, 다음글 보기
    
  #### 2. 관리자 페이지<br>
    - 메인 페이지(세계시간, 간편메뉴, 기간별 예약건수 그래프)
    - 공지사항 등록, 수정 및 삭제
